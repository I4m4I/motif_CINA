```text
artifacts/results/
  jango/
    seed43/
    seed44/
    ...
  calcium/
    seed42/
    seed46/
    ...
  mice_lick/
    seed43/

```

The exact seed folder contents are documented in `scripts/*_experiment_params.md`.
